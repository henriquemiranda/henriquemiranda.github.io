yambo-py

Published: Sun 20 September 2015
By Henrique Miranda

In Materials simulation.

Lattely I've started using Yambo to calculate GW corrections and optical absorptions spectra. As any of you that did this before can testify, to find the right convergence criteria can be a rather daunting task. To help with this I created yambopy, a pytohn library to help to make convergence tests and analyse them.

This library is still in and early development phase, feel free to contribute with suggestions, patches and bug reports.
